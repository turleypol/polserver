testsmtp.src

[Command] testsmtp

Purpose
A quick command to be able to test SMTP email routing to a specific email address. After running the command you'll be prompted for an email address to send to.
This then triggers the SendEmail function using some generic settings.
Any error messages are sent as a private message to the mobile that called the command.
Parameters
-None- Maybe later down the track i'll add it so you can just type .testsmtp emailaddress.
