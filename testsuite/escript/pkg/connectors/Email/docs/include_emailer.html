emailer.inc

SendEmail( ToAddress, Subject, Body, Modifiers:=0)

Purpose
Sends an email to address or addresses based on information given to the function.
Modifiers is the only non-required parameter.
Parameters
ToAddress String - The email address the email is being sent to.
Subject String - The subject line of the email body.
Body String - The text to be put into the body.
Modifiers Struct - A struct of modifiers that SMTP connections support.
Optional Modifiers(Struct)
FromEmail String - To define an email address that the email is from otherwise default from the settings file will be used.
ToContactName String - To define a name which will be displayed next to the email in the recipients mail client.
CC String - To define carbon copy emails to be sent, carbon copy email addresses can be seen by all recipients.
BCC String - To define blind carbon copy emails to be sent, the same as above only other receipients can't see who was BCC'ed.
ReplyTo String - To define a different email address to be replied to. This is different to FromEmail, when a client replies to an email sent this email address would be used in place of FromEmail.
Returns
Success - Integer 1
Failure - Errortext

ValidateEmail( Address )

Purpose
Checks a string to make sure it contains an @ symbol and a full stop.
Parameters
Address String - The email address that is to be checked for validity.
Returns
Success - Integer 1
Failure - Integer 0 - (Later will update to error message)
