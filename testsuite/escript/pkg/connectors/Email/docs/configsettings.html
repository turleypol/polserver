Config Settings


Settings.cfg Settings
 
   
'Settings' Entries
Settings used by the package for default processing.
Format for them is:
<SettingName> <Value>

Example:
Host 127.0.0.1
Port 25
 
Parameters
Host The IP address of the SMTP server.
Port The port to be used for SMTP connection(normally port 25).
FromEmail The default email address emails will be sent from if one is not defined by the script.
LoggingLevel The depth of logs created by the package.
Log Level Depth
0 - No Logs Generated
1 - Errors only are recorded to logfile.
2 - Minor information recorded to log file. To and Subject. In addition to level 1.
3 - Medium information recorded to log file. All outbound information. In addition to level 2.
4 - Maximum information recorded to log file. All outbound and inbound information. In addition to level 3.
10 - The same as level 4 only it also prints to console
